# 报价计算器

一个用于深圳金大全 PI / CI / Quotation 的网页版报价工具。

## 功能

- 输入客户、单号、出货国家、付款方式、交期。
- 维护货品明细：Description、QTY(KG)、Unit Price(EXW)、Subtotal。
- 输入本地费用、海运费、银行手续费、其他费用。
- 支持 EXW / FOB / CFR / CIF 报价口径。
- 保险可按费率计算，也可勾选船公司赠送 / 不计费。
- 按公司现有模板生成 PI、CI、QUOTATION 三种 Excel 单据。

## 使用

```bash
npm install
npm run dev
```

打开终端提示的本地网址即可使用。
